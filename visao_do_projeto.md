# Product Vision

## 1. Projeto

O nome do sistema será DoCarreto, é um sistema para unir pessoas com demanda e pessoas dispostas prestarem serviços.

## 2. Objetivo do Projeto

O objetivo desse projeto é unir pessoas com necessidades de serviços de transporte de grandes volumes com prestadores de serviços de carreto.

## 3. Problema que o sistema resolve

Dificuldade em encontrar prestadores de serviços hábeis, dificuldade dos prestadores para encontrar serviço.

## 4. Público Alvo

Prestadores de serviços e solicitantes com demandas para transporte de grande volume.

## 5. Objetivos do Produto

Permitir a comunicação entre o prestador de serviço e anunciante, anunciar serviço, permitir a busca por serviço.
## 6. Principais Funcionalidades
Cadastrar anúncios de trabalho, cadastrar usuários, chat entre as partes envolvidas, busca por trabalho
## 7. Critérios de Sucesso
O critério de sucesso é o anunciante encontrar o serviço desejado e aumentar a demanda de trabalho do prestador de serviço.
